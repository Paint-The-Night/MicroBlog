Design Princnples

The 3 Major Design Principles for this blog are. Portable & Tiny

No Ads

This website will never include ads

No Frameworks

No Web Building Frameworks will be used for the website. It will all be pure HTML/CSS/Javascript

No Server Side Code

A minimal amount of code will be used as this micro blog is supposed to emphasis text, writing and ideas. If simple script is needed then it should be done in pure javascript on the client side.

Open Source

Source Code for this entire website will be published

No Cookies

Cookies will not be used for this website. This will compromise some features

Tiny Size

Keeping the file size as tiny as possible. Last Recorded Size: 176Kb

Portable

As much as possible keep information on this blog self-contained.Links can be used when appropriate but never rely on them to be accessible.
